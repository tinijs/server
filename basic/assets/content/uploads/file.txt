Put any public file in this folder.

You can organize the content in anyway you see fit.
+ By date: 2024/01, 2024/02, ...
+ By types: documents, videos, audios, ...
+ By purposes: project-1, topic-2, ...

Files can be accessed using the path format: /tini-content/uploads/file.txt
Inside templates, use: {{pathGlobalUploads}}/file.txt

Alternative, you can put any "uploads" folders inside any item's folder, for example: /tini-content/posts/a-sample-post/uploads/file.txt
